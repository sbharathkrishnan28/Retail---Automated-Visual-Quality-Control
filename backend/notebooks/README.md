# Notebooks Directory

This folder contains Jupyter Notebooks (`.ipynb`) used by the Machine Learning Engineering team for experimentation and exploratory data analysis (EDA).

## Usage
* `01_data_exploration.ipynb` - Visualizing PCB defect distributions and testing augmentation strategies.
* `02_model_prototyping.ipynb` - Experimenting with ResNet50 vs MobileNetV2 using Transfer Learning.
* `03_gradcam_testing.ipynb` - Prototyping and verifying the visual explainability heatmaps before migrating the logic to production Python scripts.
